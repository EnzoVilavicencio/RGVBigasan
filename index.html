🚀 How to Deploy This POS Online (Free)

⚠️ Important: This POS uses your browser's local storage. Data is saved on the device/browser you use. For multi-device access, use the same browser on the same device, or consider the GitHub Pages method below which still uses local storage per device.

Option A: GitHub Pages (Recommended — Free Forever)
1
Go to github.com and create a free account if you don't have one.
2
Click New Repository. Name it tindahan-pos. Set it to Public. Click Create.
3
Save this webpage as index.html on your computer (File → Save Page As in your browser).
4
In your new GitHub repo, click Add file → Upload files. Upload index.html. Click Commit.
5
Go to Settings → Pages. Under "Source", select main branch, root folder. Click Save.
6
After a minute, your POS will be live at: https://YOUR-USERNAME.github.io/tindahan-pos 🎉

Option B: Netlify Drop (Fastest — 30 seconds)
1
Save this webpage as index.html on your computer.
2
Go to netlify.com/drop — no account needed!
3
Drag and drop your index.html file into the Netlify Drop zone. Done! You'll get a free URL instantly.

💡 Tips
• Use Export CSV regularly to back up your data.
• For best experience, use this on a tablet or laptop at your store counter.
• Bookmark the deployed URL on your phone/tablet for quick access.
• Data is stored in the browser — clearing browser data will erase it. Export CSV often!
